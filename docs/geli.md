**GELI** - [FreeBSD](freebsd.md) Cryptographic
[GEOM](geom.md) class written by Pawel Jakub Dawidek. Supports
various ciphers: [AES](aes.md), [Blowfish](blowfish.md)
and [3DES](3des.md). Supports hidden volumes and Pre-Boot
Authentification.
